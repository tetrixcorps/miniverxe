# Using Explorium MCP to Generate Leads

Since the Explorium MCP server is configured in Cursor, I'll query it directly to generate leads.

## Healthcare Leads Query
Find healthcare companies with 50-500 employees that use Epic or Cerner EHR systems and need patient communication solutions. Get their company details and find IT directors or operations managers with contact information.

## Fleet Management Leads Query  
Find fleet management companies with 100-500 vehicles that need real-time tracking and route optimization. Get fleet managers and operations directors with emails and phone numbers.

## Construction Leads Query
Find general contractors with 50-500 employees that need project management software and safety compliance monitoring. Get project managers and safety officers with contact information.
